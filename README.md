#### Antimony Web Editor

I led development of a browser-based editor for the Antimony modeling language.

My work included:

* Planning features and project direction
* Making frontend and technical design decisions
* Building the application with React and TypeScript
* Coordinating development and reviewing progress
* Presenting the project at the COMBINE conference

[View the Antimony Web Editor](https://github.com/sys-bio/AntimonyEditor)

#### VSCode-Antimony

I helped build a Visual Studio Code extension that gives researchers a better environment for writing Antimony models.

The extension reduces manual setup and provides language tooling directly inside VS Code. It has reached more than 1,000 users through the VS Code Marketplace.

[View VSCode-Antimony](https://github.com/sys-bio/vscode-antimony)
